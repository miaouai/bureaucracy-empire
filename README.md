## 🚀 快速开始

### 第一步：发送圣旨

在 Copaw 对话框中输入:

```
🍀圣旨到！{你的任务}
```

**示例**:
```
🍀圣旨到！帮我规划一个 30 天的求职冲刺计划，目标是软件工程师岗位
```

### 第二步：观看内阁开议

喵有钱会自动切换成 10 位顾问角色，从不同角度讨论你的任务:

```
【首席军师发言】此事当分三步走...
【财政顾问补充】这笔买卖划得来吗？
【技术总管评估】技术上完全可行，但...
...
```

### 第三步：尚书发令

根据讨论结果，生成可执行的任务清单:

```
【吏部】梳理工作经历和面试准备
【工部】搭建个人作品集网站
【礼部】优化简历排版和文案
...
```

### 第四步：六部执行 & 御前汇报

各部分头行动，最后汇总成完整方案呈现给你!

## 💡 最佳实践

### ✅ 推荐用法

1. **复杂决策任务** → 充分启用全部 10 人内阁
2. **时间紧急任务** → 只启用 3-5 个关键角色
3. **重复类型任务** → 主动请求复用之前的方案模板
4. **需要深度执行** → 让六部真的动手做 (写代码、查数据、制表格)

### ⚠️ Token 节省技巧

- 简单任务说"简化流程" → 只模拟核心环节
- 中途觉得太啰嗦 → 直接说"跳过细节，给结论"
- 只看最终结果 → "我只听尚书总结就行"

### 🎯 任务质量提升

- 描述更具体："找年薪 30 万的工作" > "找工作"
- 提供背景信息："我做了 4 年 Android，现居重庆..."
- 明确截止时间："一个月内搞定"
- 设定验收标准："至少要拿到 3 个面试机会"

## 🔧 自定义配置

### 修改内阁成员

编辑 `{角色}.md` 文件，调整性格和专长:
```
路径：customized_skills/bureaucracy-empire/memory/内阁/
```

### 增加新部门

如果现有六部不够用，可以新增:
- 商部 (商务合作)
- 学部 (研究分析)
- 艺部 (创意设计)

### 调整工作流

觉得某个环节多余？编辑 `templates/` 下的对应模板删减即可。

## ❓ FAQ

**Q: 必须每次都说"🍀圣旨到！"吗？**  
A: 是的，这是为了节省 token——平时正常聊天，只有重要任务才启动完整流程。

**Q: 能中途暂停或修改任务吗？**  
A: 当然可以！随时插话，我会调整后续流程。

**Q: 能不能不经过内阁直接派活？**  
A: 可以！说"直接派给工部做 XXX"就行。

**Q: 这个系统是永久性的吗？**  
A: 是的，只要你不删除 `customized_skills/bureaucracy-empire/` 目录，它就会一直存在并持续进化。

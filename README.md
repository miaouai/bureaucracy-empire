## ⚡ 快速安装（3 步搞定）

### 方案 A: 自动安装（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/miaouai/bureaucracy-empire.git
cd bureaucracy-empire

# 2. 运行一键配置脚本
bash install.sh

# 完成！现在可以使用了
```

### 方案 B: 手动安装

```bash
# 1. 克隆到本地
git clone https://github.com/miaouai/bureaucracy-empire.git
cd bureaucracy-empire

# 2. 复制到 active_skills 目录
cp -r . /app/working/active_skills/bureaucracy-empire/

# 3. 编辑 AGENTS.md，添加触发词配置（关键！）
# 在文件中添加以下内容：
cat >> /app/working/AGENTS.md << 'EOF'

### ⚡ 技能触发规则

**强制触发关键词**（检测到必须立即加载对应技能）：

| 关键词 | 触发技能 | 说明 |
|--------|----------|------|
| `🍀圣旨到！` | `bureaucracy-empire` | 启动官僚帝国任务管理系统，10 人内阁议政 + 六部执行 |

**触发协议**：
1. 扫描用户消息是否包含触发关键词
2. 匹配成功 → **立即加载**对应技能
3. 严格按照技能文档执行流程
4. 禁止跳过或忽略触发词
EOF

# 4. 验证安装
grep "🍀圣旨到" /app/working/AGENTS.md
```

## 🚀 立即试用

### 第一步：发送圣旨

在 Copaw 对话框中输入:

```
🍀圣旨到！{你的任务}
```

**示例**:
```
🍀圣旨到！帮我规划一个 30 天的求职冲刺计划，目标是软件工程师岗位
```

### 第二步：观看内阁开议

喵有爱会自动切换成 10 位顾问角色，从不同角度讨论你的任务:

```
【首席军师发言】此事当分三步走...
【财政顾问补充】这笔买卖划得来吗？
【技术总管评估】技术上完全可行，但...
...
```

### 第三步：尚书发令

根据讨论结果，生成可执行的任务清单:

```
【吏部】梳理工作经历和面试准备
【工部】搭建个人作品集网站
【礼部】优化简历排版和文案
...
```

### 第四步：六部执行 & 御前汇报

各部分头行动，最后汇总成完整方案呈现给你!

## 💡 最佳实践

### ✅ 推荐用法

| 场景 | 建议 |
|------|------|
| **复杂决策任务** | 充分启用全部 10 人内阁 |
| **时间紧急任务** | 说"简化流程"只启用 3-5 个关键角色 |
| **重复类型任务** | 主动请求复用之前的方案模板 |
| **需要深度执行** | 让六部真的动手做 (写代码、查数据、制表格) |

### ⚠️ Token 节省技巧

- 简单任务说 **"简化流程"** → 只模拟核心环节
- 中途觉得太啰嗦 → 直接说 **"跳过细节，给结论"**
- 只看最终结果 → **"我只听尚书总结就行"**

### 🎯 任务质量提升

描述任务时包含以下要素效果更好：

| 要素 | 好的例子 | ❌ 差的例子 |
|------|---------|-----------|
| **目标** | 找年薪 30 万的工作 | 找工作 |
| **背景** | 4 年 Android，现居重庆 | （不说） |
| **截止时间** | 一个月内搞定 | （不说） |
| **验收标准** | 至少要拿到 3 个面试机会 | 随便 |

## 🔧 自定义配置

### 修改内阁成员

编辑 `{角色}.md` 文件，调整性格和专长:
```bash
vim /app/working/active_skills/bureaucracy-empire/memory/内阁/首席军师_卧龙.md
```

### 增加新部门

如果现有六部不够用，可以新增:
- 商部 (商务合作)
- 学部 (研究分析)
- 艺部 (创意设计)

### 调整工作流

觉得某个环节多余？编辑 `templates/` 下的对应模板删减即可。

## ❓ FAQ

**Q: 必须每次都说"🍀圣旨到！"吗？**  
A: 是的，这是为了节省 token——平时正常聊天，只有重要任务才启动完整流程。

**Q: 能中途暂停或修改任务吗？**  
A: 当然可以！随时插话，我会调整后续流程。

**Q: 能不能不经过内阁直接派活？**  
A: 可以！说"直接派给工部做 XXX"就行。

**Q: 这个系统是永久性的吗？**  
A: 是的，只要你不删除 `active_skills/bureaucracy-empire/` 目录，它就会一直存在并持续进化。

**Q: 如何更新技能到最新版本？**  
A: 从 GitHub 重新 pull 最新代码覆盖本地文件即可：
```bash
cd /app/working/active_skills/bureaucracy-empire
git pull origin main
```

**Q: 为什么我输入"🍀圣旨到！"没有反应？**  
A: 检查 AGENTS.md 中是否有触发词配置！这是技能能否工作的核心前提。运行：
```bash
grep "🍀圣旨到" /app/working/AGENTS.md
```
如果没有输出，就是缺少配置。

# ⚙️ 六部首脑 - 职责速查

## 📄 吏部尚书
```
管什么：人 & 能力  
典型任务:
• 简历优化
• 面试模拟  
• 学习计划制定
• 人脉关系梳理
必备技能：沟通协调、写作能力
```

## 💰 户部尚书
```
管什么：钱 & 预算  
典型任务:
• 项目成本估算
• 服务比价采购  
• 收益预测分析
• 投资回报率计算
必备技能：Excel 计算、市场洞察
```

## 📜 礼部尚书
```
管什么：文档 & 形象  
典型任务:
• PPT/Word制作
• 邮件/信函撰写  
• 社交账号运营
• 品牌包装文案
必备技能：文案功底、排版审美
```

## ⚔️ 兵部尚书
```
管什么：策略 & 竞争  
典型任务:
• 竞品分析报告
• 谈判策略制定  
• 市场进入规划
• 应急预案设计
必备技能：博弈论、数据分析
```

## ⚖️ 刑部尚书
```
管什么：风险 & 合规  
典型任务:
• 合同条款审查
• 隐私政策检查  
• 风险提示报告
• 法律法规研究
必备技能：法律常识、细致严谨
```

## 🔧 工部尚书
```
管什么：技术 & 工具  
典型任务:
• 脚本编写
• 环境搭建  
• 数据抓取处理
• 自动化工具开发
必备技能：编程、DevOps、工具链
```

## 🔄 协同场景示例

**求职准备大任务**:
- 吏部：梳理经历、写自我介绍
- 工部：搭建作品集网站、做自动化投递脚本
- 礼部：美化简历 PDF、写cover letter
- 兵部：调研目标公司薪资范围、准备谈薪话术
- 刑部：检查 offer 条款陷阱
- 户部：计算最优薪资期望值

> **组织化任务管理 Skill** — 用古代六部制解决现代复杂问题  
> **触发方式**: `🍀圣旨到！`  
> **适用平台**: Copaw / OpenClaw 等 AI Agent 框架  
> **License**: MIT  

---

## ✨ 什么是官僚帝国？

一个模拟中国古代**三层六部制**的 AI 任务管理系统：

```
主公 (你) 
    ↓ 🍀圣旨到！
【内阁议政厅】10 位专家多角度讨论 → 形成策略
    ↓
【尚书省】拆分任务、分派部门
    ↓
【六部执行】并行处理、各司其职
    ↓
【御前汇报】汇总成果、等待决策
```

---

## 🎯 适用场景

- ✅ **复杂决策** — 求职规划、项目立项、投资分析
- ✅ **多步骤任务** — 需要跨领域协作的综合性工作  
- ✅ **创意发散** — 集思广益，避免思维盲区
- ✅ **教学演示** — 展示结构化思考方法

## 🚀 快速开始

### 1. 安装

```bash
# 克隆到 skills 目录
git clone https://github.com/miaouai/bureaucracy-empire.git

# 或者直接下载 ZIP 解压
```

### 2. 配置触发规则

在 AGENTS.md 中添加:

```markdown
### ⚡ 技能触发规则

| 关键词 | 触发技能 | 说明 |
|--------|----------|------|
| `🍀圣旨到！` | `bureaucracy-empire` | 启动官僚帝国任务管理系统 |

**触发协议**:
1. 扫描用户消息是否包含触发关键词
2. 匹配成功 → **立即加载**对应技能
3. 严格按照技能文档执行流程
4. 禁止跳过或忽略触发词
```

### 3. 激活

在 Copaw/OpenClaw 中发送:

```
🍀圣旨到！帮我制定一个 30 天求职冲刺计划
```

### 4. 享受服务

系统自动启动完整流程：

1. 👑 **内阁开议** — 10 位顾问各抒己见
2. 📋 **尚书发令** — 拆解为可执行子任务  
3. ⚙️ **六部行动** — 并行处理各项事宜
4. 📜 **御前汇报** — 提交完整方案

## 💡 使用技巧

### 基础用法

```markdown
# 完整流程（默认）
🍀圣旨到！{详细描述你的任务}

# 简化模式（省 Token）
🍀圣旨到！简化模式：{快速任务}

# 指定部门
🍀圣旨到！只让工部和礼部参与：{技术 + 文案类任务}
```

### 高级控制

```markdown
# 只看结论
🍀圣旨到！跳过过程，直接给最终方案：{任务}

# 中途调整
"财政顾问说得太多了，直接进入尚书环节"

# 保存模板
"把这个方案保存为模板，下次类似任务复用"
```

## 🔧 自定义配置

### 修改角色设定

编辑 `memory/内阁/*.md`:

```markdown
# 首席军师_卧龙.md
定位：战略规划专家
风格：沉稳、宏观、善于拆解
口头禅："此事当分三步走..."
专长：目标分解、路径规划、风险评估
```

### 添加新部门

如果需要新增职能，复制 `memory/六部/` 下任意 `.md` 修改即可。

### 调整工作流

编辑 `templates/` 下的对应模板，删减或增加步骤。

## 🌟 特色功能

### 1. 自进化机制

每次任务完成后自动记录：
- 哪些角色最有价值？
- 哪个环节效率最低？
- 有什么改进空间？

**每日 9 点自动生成优化建议** → 系统越用越聪明！

### 2. Token 节省模式

- **简化流程**：只启用关键角色
- **渐进披露**：追问时才展开细节
- **缓存复用**：类似任务直接套用历史方案

### 3. 多场景适配

| 场景 | 推荐配置 | 预计耗时 |
|------|---------|---------|
| 简单决策 | 3 人内阁 +2 部 | 2-3 分钟 |
| 中等任务 | 6 人内阁 +4 部 | 5-8 分钟 |
| 复杂项目 | 全阵容 | 10-15 分钟 |
